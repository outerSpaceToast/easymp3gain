Anfordrungen:
easyMP3Gain benötigt ein Linux-basierendes Betriebssystem mit X11 und die installierten Programme mp3gain und/oder vorbisgain und/oder aacgain. Wenn ihr Betriebssystem keine mp3gain-Pakete bereitstellt, laden Sie es manuell herunter. Stellen Sie sicher, dass sich mp3gain/vorbisgain/aacgain oder ein Link darauf im Verzeichnis "/bin" befindet.
Installieren (Debian/xUbuntu/LinuxMint)
  1. Lade das ".deb" Paket von easyMP3Gain herunter
  2. sudo dpkg -i [Paketname mit Pfad]
Installieren (openSuSE/Fedora/RedHat)
  1. Lade das ".rpm" Paket von easyMP3Gain herunter
  2. sudo rpm -i [Paketname mit Pfad]
Installieren (.tar.gz)
  1. Lade das ".tar.gz" Paket von easyMP3Gain herunter
  2. Entpacken Sie den Inhalt des Pakets in ein gew√ºnschtes Verzeichnis.
Starten (.deb / .rpm):
  1. Starten Sie das Programm, indem Sie auf den entsprechenden Eintrag im Startmenü klicken.
Starten (.tar.gz):
  1. Wechseln Sie in das 'easyMP3Gain'-Verzeichnis, in das Sie die Dateien entpackt haben.
  2. Starten Sie easyMP3Gain mit dem Befehl
    "./easymp3gain"
